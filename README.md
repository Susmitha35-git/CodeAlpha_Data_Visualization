# 🦠 COVID-19 Global Data Analysis Dashboard (2020–2023)

### CodeAlpha Data Analytics Internship – Dashboard Project

This project presents a comprehensive analysis of global COVID-19 trends between 2020 and 2023 using Python-based data visualization techniques. The dashboard explores case counts, death statistics, vaccination progress, fatality rates, and regional impacts across major countries affected by the pandemic.


## 📌 Project Overview

The COVID-19 pandemic had a significant impact worldwide, affecting millions of people across different countries and regions. This project analyzes COVID-19 statistics and visualizes key trends to better understand the spread of the virus, mortality rates, vaccination progress, and recovery patterns.

The project uses data visualization and exploratory analysis techniques to transform raw pandemic data into meaningful insights.


## 🎯 Project Objectives

* Analyze global COVID-19 case trends.
* Compare total cases and deaths across countries.
* Study vaccination progress among nations.
* Identify countries with the highest fatality rates.
* Visualize pandemic waves over time.
* Examine regional distribution of COVID-19 cases.
* Generate insights using interactive dashboard visualizations.


## 📊 Dataset Information

The dataset contains COVID-19 statistics collected from multiple countries between 2020 and 2023.

### Features Used

| Feature | Description |
|----------|-------------|
| Country | Country Name |
| Total Cases | Total confirmed COVID-19 cases |
| Total Deaths | Total deaths due to COVID-19 |
| Cases per Million | Cases normalized by population |
| Deaths per Million | Deaths normalized by population |
| Vaccination Rate | Percentage of vaccinated population |
| New Cases | Weekly reported new cases |
| Case Fatality Rate (CFR) | Percentage of deaths among confirmed cases |
| Region | Geographic region |

## 🔑 Key Findings

### 🦠 Global Case Analysis

* The United States reported the highest number of confirmed COVID-19 cases.
* India ranked second in total reported cases.
* Several European countries experienced significant infection rates.

### ⚰️ Mortality Analysis

* The United States recorded the highest number of COVID-19 deaths.
* Brazil and India also experienced substantial mortality impacts.
* Death rates varied considerably across countries.

### 📈 Pandemic Waves

* Multiple global waves were observed between 2020 and 2022.
* The Delta and Omicron variants contributed to major spikes in cases.
* New case counts declined significantly during 2023.

### 💉 Vaccination Progress

* Argentina achieved the highest vaccination coverage among the analyzed countries.
* Most countries surpassed the 70% vaccination target.
* Increased vaccination rates corresponded with reduced fatality rates.

### ⚠️ Fatality Rate Analysis

* Mexico exhibited the highest Case Fatality Rate (CFR).
* Countries with stronger healthcare responses generally reported lower fatality rates.

### 🌍 Regional Impact

* Europe and the Americas accounted for the largest share of total cases.
* Asia contributed a significant portion of global infections.
* Africa and Oceania represented smaller shares of total reported cases.

3. Generate visualizations.
4. Create the dashboard.
5. Save the output image as `covid_dashboard.png`.
